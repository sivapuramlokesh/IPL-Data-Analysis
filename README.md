# IPL Data Analysis Dashboard (2008–2025)

## 📊 Overview

This project analyzes IPL data from 2008 to 2025 using Power BI. It provides insights into team performance, player statistics, and match trends.

## 🚀 Features

* Interactive season-wise dashboard
* KPIs: Matches, Runs, Wickets, Boundaries
* Orange Cap & Purple Cap analysis
* Team points table visualization

## 🛠️ Tools Used

* Power BI
* Excel
* DAX

## 📈 Key Insights

* Team performance varies across seasons
* Player stats highlight consistency and impact
* Match trends show evolution of gameplay
